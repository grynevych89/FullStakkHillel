Набір невеликих сторінок для практики з display, box-sizing, float/clear, overflow та position.

Сторінки:

  1. Блокові та інлайнові елементи
  2. normalize.css: спостереження та reset.css: спостереження

  4. float + clear
  5. overflow
  6. position: static/relative/absolute/fixed/sticky

Порада:

Відкривайте кожну сторінку у двох вікнах/вкладках і змінюйте CSS у DevTools, щоб одразу бачити різницю. Для пункту 3 використані окремі файли normalize.css та reset.css, вже підключено у відповідних сторінках.
